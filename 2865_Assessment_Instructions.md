## 1. Assessment Instructions

Welcome to the Module 11 Assessment for **Full Stack .NET Development with C#, ASP.NET Core, and Angular**. This assessment is designed to test your understanding of implementing client-side authentication and authorization logic in Angular.

**Topics Covered:**

  * Client-Side Authentication Flow
  * Designing the Angular authentication flow
  * Creating login and registration components
  * Submitting credentials to the backend API
  * Handling the JWT response
  * Storing the JWT (localStorage vs. sessionStorage vs. HttpOnly cookies)
  * Implementing logout functionality
  * Creating an `AuthService`
  * Implementing login and registration methods in the service
  * Storing the JWT in `localStorage`
  * Retrieving and clearing the token
  * Creating an `AuthInterceptor` to attach the token to requests
  * Handling token expiration
  * Implementing an `AuthGuard` in Angular
  * Checking for the presence of a JWT token
  * Redirecting unauthenticated users to the login page
  * Applying the `AuthGuard` to protected routes
  * Creating a mock user profile page



**Tips for Success:**

  * Review all lessons in Module 11 thoroughly.
  * Pay close attention to the practical implementation details, especially regarding `AuthService`, `AuthInterceptor`, and `AuthGuard`.
  * Understand the security implications of different token storage mechanisms.
  * Consider how JWTs are generated, validated, and used in a full-stack application.
  * Read each question carefully before selecting your answer.



Good luck!
