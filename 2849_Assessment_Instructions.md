## 1. Assessment Instructions

**Welcome to the Module 7 Assessment!**

This assessment is designed to test your understanding of the key concepts covered in Module 7: Angular Services and HTTP Client. You will be evaluated on your knowledge of creating and utilizing Angular services, understanding dependency injection, working with Observables and RxJS, and making HTTP requests to interact with backend APIs.

**Topics Covered:**

  * Angular Services and Dependency Injection
  * The `@Injectable()` decorator and service registration
  * Singleton pattern for services
  * Introduction to Observables and RxJS
  * Subscribing to Observables
  * Common RxJS operators (`map`, `filter`)
  * Error handling in Observables
  * Using Angular's `HttpClient` for GET, POST, PUT, DELETE requests
  * Handling API responses and errors
  * Consuming an ASP.NET Core API



**Tips for Success:**

  * **Read Carefully:** Pay close attention to the wording of each question.
  * **Review Module Content:** If you are unsure about a topic, refer back to the lesson materials.
  * **Consider Best Practices:** Many questions relate to recommended Angular development practices.
  * **Time Management:** Allocate your time wisely across all questions.



Good luck!
